Tock OS - Secure Embedded Operating System

Stanford & MIT Research, Google, Western Digital

Tock is a secure embedded operating system for running multiple concurrent, mutually distrustful applications on low-memory microcontrollers. Used in production by Google (OpenSK security keys) and Western Digital.

Architecture
ARM Cortex-M
Isolation
MPU-based
Status
Production

Key Features:

  • Memory Protection Unit (MPU) isolation between apps
  • Capsules for kernel extension without recompilation
  • Cooperative scheduling with preemption
  • Support for multiple concurrent applications
  • Low memory footprint (< 10KB kernel)

Production Deployments:

  • OpenSK: Google's open-source FIDO2 security key
  • Western Digital: Storage device firmware
  • Research: Multiple academic security projects
Rust ARM Cortex-M4 RISC-V MPU FIDO2

🎓 Learning from Real Projects

Common Patterns in Production Systems

  • Embassy async runtime: Increasingly preferred over RTIC for new projects
  • defmt for logging: Standard choice for efficient debugging
  • probe-rs: Dominant debugging and flashing tool
  • embedded-hal traits: Enables portable driver development
  • Device-agnostic drivers: Reusable across different MCUs
  • Workspace organization: Separate HAL, drivers, and application

💡 Key Takeaways

Production Reality: Embedded Rust has moved beyond experimentation to powering mission-critical systems in aerospace, automotive, security, and IoT domains.

Safety & Reliability: Organizations choose Rust for systems where failures are unacceptable - satellites, automotive safety systems, cryptographic hardware, and industrial control.

Performance: Rust delivers C/C++ level performance with modern abstractions, as demonstrated by high-throughput networking (Cloudflare) and real-time robotics applications.

Ecosystem Maturity: The availability of qualified toolchains (Ferrocene), production RTOSes (Tock, Hubris), and widespread HAL support indicates a mature, production-ready ecosystem.

Future Outlook: With increasing vendor support, growing community, and proven production deployments, embedded Rust adoption is accelerating across all sectors.
