Cargo & Package Management

What is Cargo?

Cargo is Rust's official package manager and build system. It handles many tasks including:

  • 🔨 Building your code with cargo build
  • ▶️ Running your project with cargo run
  • ✅ Testing your code with cargo test
  • 📦 Managing dependencies from crates.io
  • 📚 Generating documentation with cargo doc
  • 🚀 Publishing packages to crates.io

Cargo comes bundled with Rust when you install it using rustup. It's the primary tool you'll use for Rust development.
