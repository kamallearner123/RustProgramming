📦 Rust Installation Guide

📌 Overview: This comprehensive guide will walk you through installing Rust on your operating system. Follow the steps carefully for your specific platform.

Installing Rust on Windows

1 Choose Your Toolchain

Windows offers two toolchain options:

  • MSVC (Recommended): Native Windows toolchain, better integration
  • GNU (MinGW-w64): Unix-like toolchain, smaller size
For most users, MSVC is recommended as it provides better Windows integration and is the default choice.
2 Install Visual C++ Build Tools (MSVC only)

If choosing MSVC, you'll need the Microsoft C++ Build Tools:

  • Download from: Visual Studio Downloads
  • Select "Build Tools for Visual Studio 2022"
  • In the installer, check "Desktop development with C++"
  • This is approximately 6-8 GB download
⚠️ Alternative: If you don't want to install Visual Studio, rustup will offer to install the build tools automatically during Rust installation.
3 Download Rustup Installer

Visit https://rustup.rs and download the installer:

  • Click "Download Rustup-init.exe (64-bit)"
  • Save the file to your Downloads folder
  • File size: approximately 8-10 MB
Alternative Method: Using Windows Package Manager (winget):
winget install Rustlang.Rustup
4 Run the Installer

Execute rustup-init.exe:

  • Double-click the downloaded file
  • A command prompt window will open
  • You'll see installation options
  • Press 1 and Enter for default installation
  • Or press 2 to customize installation
Welcome to Rust! This will download and install Rust toolchain. Current installation options: default host triple: x86_64-pc-windows-msvc default toolchain: stable (default) profile: default modify PATH variable: yes 1) Proceed with installation (default) 2) Customize installation 3) Cancel installation >
5 Wait for Installation

The installer will:

  • Download the Rust toolchain (approximately 200-300 MB)
  • Install rustc, cargo, and other tools
  • Configure your PATH environment variable
  • Set up default profile
Installation typically takes 5-15 minutes depending on your internet speed.
6 Restart Your Terminal

After installation completes:

  • Close all PowerShell or Command Prompt windows
  • Open a new PowerShell or Command Prompt
  • This ensures the PATH is updated
7 Verify Installation

Check that Rust is installed correctly:

# Check Rust compiler version rustc --version # Check cargo version cargo --version # Check rustup version rustup --version
✅ Expected Output: 
rustc 1.82.0 (f6e511eec 2024-10-15)
cargo 1.82.0 (8f40fc59f 2024-08-21)
rustup 1.27.1 (54dd3d00f 2024-04-24)
⚠️ Common Issues:
  • Missing MSVC: If you see linker errors, install Visual C++ Build Tools
  • PATH not updated: Restart your terminal or PC
  • Antivirus interference: Temporarily disable if installation hangs

Installing Rust on macOS

1 Install Xcode Command Line Tools

Rust requires a C compiler and linker, provided by Xcode CLI tools:

xcode-select --install
  • A dialog box will appear asking to install
  • Click "Install" and agree to the license
  • Download size: approximately 500 MB - 1 GB
  • Installation takes 5-20 minutes
Already Installed? If you see "command line tools are already installed", you're good to go!
2 Download and Run Rustup

Open Terminal and run the rustup installer:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

This command:

  • Downloads the rustup installation script securely
  • Executes the installer
  • Uses TLS 1.2 for secure connection
3 Choose Installation Options

The installer will present options:

Current installation options: default host triple: x86_64-apple-darwin # or: aarch64-apple-darwin (M1/M2/M3) default toolchain: stable (default) profile: default modify PATH variable: yes 1) Proceed with installation (default) 2) Customize installation 3) Cancel installation >

Press 1 and Enter for default installation.

4 Wait for Installation

Rustup will:

  • Download Rust toolchain (200-300 MB)
  • Install to ~/.cargo and ~/.rustup
  • Update ~/.zshrc or ~/.bash_profile
  • Set up PATH environment variable
You'll see: "Rust is installed now. Great!" when complete.
5 Configure Your Shell

Apply the PATH changes:

# For zsh (default on macOS Catalina+): source ~/.zshrc # For bash: source ~/.bash_profile # Or simply close and reopen Terminal
6 Verify Installation

Confirm Rust is working:

# Check versions rustc --version cargo --version rustup --version # Check architecture (should show aarch64 for M-series) rustc --version --verbose | grep host
✅ For Apple Silicon: Should show aarch64-apple-darwin
✅ For Intel Macs: Should show x86_64-apple-darwin
🍎 Apple Silicon Notes:
  • Rust has native ARM64 support for M1/M2/M3/M4 chips
  • No Rosetta 2 translation needed
  • Performance is excellent on Apple Silicon

Installing Rust on Linux

1 Install Build Tools

Install a C compiler and linker first:

Ubuntu / Debian:

sudo apt update sudo apt install build-essential

Fedora / RHEL / CentOS:

sudo dnf groupinstall "Development Tools"

Arch Linux / Manjaro:

sudo pacman -S base-devel

openSUSE:

sudo zypper install -t pattern devel_basis
2 Download and Run Rustup

Execute the rustup installer:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
Alternative: Some distributions have rust in their package managers, but rustup is recommended for the latest versions and easier updates.
3 Choose Installation Options

Select installation preferences:

Current installation options: default host triple: x86_64-unknown-linux-gnu default toolchain: stable (default) profile: default modify PATH variable: yes 1) Proceed with installation (default) 2) Customize installation 3) Cancel installation >

Press 1 for default installation.

4 Configure Your Shell

After installation, update your PATH:

# For bash: source ~/.bashrc # For zsh: source ~/.zshrc # For fish: source ~/.config/fish/config.fish # Or open a new terminal
5 Install Additional Dependencies (Optional)

Some Rust crates require additional system libraries:

Ubuntu / Debian:

sudo apt install pkg-config libssl-dev

Fedora:

sudo dnf install pkgconf openssl-devel

Arch:

sudo pacman -S pkgconf openssl
6 Verify Installation

Confirm everything is working:

# Check versions rustc --version cargo --version rustup --version # Check installation location which rustc which cargo
✅ Expected Output: 
rustc 1.82.0 (f6e511eec 2024-10-15)
cargo 1.82.0 (8f40fc59f 2024-08-21)
/home/username/.cargo/bin/rustc
⚠️ Common Issues:
  • curl not found: Install curl: sudo apt install curl
  • Linker errors: Install build-essential or equivalent
  • Permission denied: Don't use sudo with rustup installer

Post-Installation Steps (All Platforms)

Update Rust

Keep your Rust installation up to date:

rustup update

Install Additional Components

Add useful components:

# Install rustfmt (code formatter) rustup component add rustfmt # Install clippy (linter) rustup component add clippy # Install rust-src (source code for standard library) rustup component add rust-src

Configure Your Editor

Install extensions for your preferred editor:

  • VS Code: Install "rust-analyzer" extension
  • IntelliJ/CLion: Install "Rust" plugin
  • Vim/Neovim: Install rust.vim and LSP plugins

🎉 Installation Complete!

You're ready to start learning Rust! Proceed to the verification page to test your setup.
