🔀 Control Flow in Rust

📌 Key Concept: Control flow constructs allow you to control the execution path of your program. Rust provides several control flow mechanisms: if expressions, loops (loop, while, for), and pattern matching with match.

1. if Expressions

In Rust, if is an expression (not a statement), which means it returns a value.

fn main() { let number = 7; if number < 5 { println!("condition was true"); } else { println!("condition was false"); } }

Multiple Conditions with else if

fn main() { let number = 6; if number % 4 == 0 { println!("number is divisible by 4"); } else if number % 3 == 0 { println!("number is divisible by 3"); } else if number % 2 == 0 { println!("number is divisible by 2"); } else { println!("number is not divisible by 4, 3, or 2"); } }
⚠️ Important: The condition in an if expression must be a bool. Rust will not automatically convert non-boolean types to boolean.
fn main() { let number = 3; // ❌ ERROR: expected `bool`, found integer // if number { // println!("number was three"); // } // ✅ CORRECT: explicit comparison if number != 0 { println!("number was something other than zero"); } }

2. Using if in a let Statement

Because if is an expression, you can use it on the right side of a let statement:

fn main() { let condition = true; let number = if condition { 5 } else { 6 }; println!("The value of number is: {}", number); }
⚠️ Type Consistency: All branches of an if expression must return the same type:
fn main() { let condition = true; // ❌ ERROR: `if` and `else` have incompatible types // let number = if condition { 5 } else { "six" }; // ✅ CORRECT: same types in both branches let number = if condition { 5 } else { 6 }; }

3. Repetition with Loops

Rust has three kinds of loops: loop, while, and for.

3.1 The loop Keyword

The loop keyword creates an infinite loop. Use break to exit.

fn main() { let mut counter = 0; loop { counter += 1; if counter == 10 { break; } } println!("Counter: {}", counter); }

Returning Values from Loops

fn main() { let mut counter = 0; let result = loop { counter += 1; if counter == 10 { break counter * 2; // Return value from loop } }; println!("The result is {}", result); // 20 }

Loop Labels

Use loop labels to break or continue specific loops in nested loops:

fn main() { let mut count = 0; 'outer: loop { println!("count = {}", count); let mut remaining = 10; loop { println!("remaining = {}", remaining); if remaining == 9 { break; // Break inner loop } if count == 2 { break 'outer; // Break outer loop } remaining -= 1; } count += 1; } println!("End count = {}", count); }

4. Conditional Loops with while

The while loop runs while a condition is true:

fn main() { let mut number = 3; while number != 0 { println!("{}!", number); number -= 1; } println!("LIFTOFF!!!"); }

Example: Countdown

fn main() { let mut countdown = 10; while countdown > 0 { println!("T-minus {}", countdown); countdown -= 1; } println!("Blast off! 🚀"); }

5. Looping Through Collections with for

The for loop is the most common loop type for iterating over collections:

fn main() { let a = [10, 20, 30, 40, 50]; for element in a { println!("the value is: {}", element); } }

Using Ranges

fn main() { // Count from 1 to 5 for number in 1..6 { println!("{}", number); } println!("---"); // Reverse countdown for number in (1..4).rev() { println!("{}!", number); } println!("LIFTOFF!!!"); }
Loop Type When to Use Example
loop Infinite loop or when you need to return a value loop { if done { break; } }
while Loop while condition is true while count < 10 { count += 1; }
for Iterate over collections or ranges for i in 0..10 { println!("{}", i); }

6. break and continue

break Statement

Exits the current loop:

fn main() { let mut count = 0; loop { count += 1; if count == 5 { break; // Exit the loop when count reaches 5 } println!("Count: {}", count); } }

continue Statement

Skips the rest of the current iteration and moves to the next:

fn main() { for number in 1..6 { if number == 3 { continue; // Skip printing 3 } println!("{}", number); } }

7. Pattern Matching with match

The match expression is a powerful control flow construct for pattern matching:

fn main() { let number = 7; match number { 1 => println!("One!"), 2 | 3 | 5 | 7 | 11 => println!("This is a prime"), 13..=19 => println!("A teen"), _ => println!("Ain't special"), } }

match with Expressions

fn main() { let boolean = true; let binary = match boolean { false => 0, true => 1, }; println!("{} -> {}", boolean, binary); }

Matching Tuples

fn main() { let pair = (0, -2); match pair { (0, y) => println!("First is `0` and `y` is `{:?}`", y), (x, 0) => println!("`x` is `{:?}` and last is `0`", x), _ => println!("It doesn't matter what they are"), } }
⚠️ Exhaustiveness: Rust requires match expressions to be exhaustive. You must handle all possible cases or use _ as a catch-all pattern.
fn main() { let number = 4; // ❌ ERROR: non-exhaustive patterns // match number { // 1 => println!("one"), // 2 => println!("two"), // } // ✅ CORRECT: all cases handled match number { 1 => println!("one"), 2 => println!("two"), _ => println!("something else"), } }

8. if let Expression

if let is a concise way to handle values that match one pattern while ignoring the rest:

fn main() { let favorite_color: Option<&str> = Some("blue"); if let Some(color) = favorite_color { println!("Your favorite color is {}", color); } }

if let vs match

// Using match fn main() { let config_max = Some(3u8); match config_max { Some(max) => println!("The maximum is {}", max), _ => (), } } // Using if let (more concise) fn main() { let config_max = Some(3u8); if let Some(max) = config_max { println!("The maximum is {}", max); } }

9. Control Flow Examples

Example 1: FizzBuzz

fn main() { for n in 1..=30 { match (n % 3, n % 5) { (0, 0) => println!("FizzBuzz"), (0, _) => println!("Fizz"), (_, 0) => println!("Buzz"), _ => println!("{}", n), } } }

Example 2: Menu Selection

fn main() { let choice = 2; match choice { 1 => println!("Start new game"), 2 => println!("Load saved game"), 3 => println!("View settings"), 4 => println!("Exit"), _ => println!("Invalid choice!"), } }

Example 3: Grade Calculator

fn get_grade(score: u32) -> char { match score { 90..=100 => 'A', 80..=89 => 'B', 70..=79 => 'C', 60..=69 => 'D', _ => 'F', } } fn main() { let scores = [95, 87, 72, 55, 90]; for score in scores { let grade = get_grade(score); println!("Score {} = Grade {}", score, grade); } }

Example 4: Nested Loops with break

fn main() { 'outer: for x in 1..=5 { for y in 1..=5 { if x * y == 10 { println!("Found: {} * {} = 10", x, y); break 'outer; } } } }

Example 5: Sum with Loop

fn main() { let numbers = [1, 2, 3, 4, 5]; let mut sum = 0; for number in numbers { sum += number; } println!("Sum: {}", sum); }

Example 6: Find First Even Number

fn main() { let numbers = [1, 3, 7, 8, 9, 12]; for number in numbers { if number % 2 == 0 { println!("First even number: {}", number); break; } } }

10. Best Practices

✅ Do's:

  • Use for loops for iterating over collections
  • Use match for complex pattern matching
  • Use if let when you only care about one pattern
  • Label nested loops for clarity
  • Keep conditions simple and readable

❌ Don'ts:

  • Don't use while to iterate over arrays (use for instead)
  • Don't forget the _ catch-all in match
  • Don't rely on non-boolean values in if conditions
  • Avoid deeply nested loops when possible

💡 Key Takeaways

  • if is an expression that returns a value
  • Conditions must be bool type (no automatic conversion)
  • Three loop types: loop (infinite), while (conditional), for (iteration)
  • break exits loops, continue skips to next iteration
  • match is powerful for pattern matching and must be exhaustive
  • if let is syntactic sugar for matching a single pattern
  • Use loop labels ('label:) for nested loop control
  • Prefer for loops over while for collection iteration
