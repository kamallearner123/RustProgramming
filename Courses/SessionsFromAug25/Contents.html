Course Curriculum (30 Hours)

  • Why Rust for Embedded & Automotive? Safety, Concurrency, Performance.
  • Setting up the Development Environment (Rustup, Cargo, VS Code).
  • Your First Embedded Program: "Hello, World!" on a microcontroller.
  • Rust Syntax Essentials: Variables, Data Types, Functions, and Control Flow.
  • Cargo Deep Dive: Managing dependencies and build configurations.
  • The Ownership Model: Ownership, Borrowing, and Lifetimes.
  • Data Structures: Structs, Enums, and powerful Pattern Matching.
  • Robust Error Handling: Mastering `Result` and `Option`.
  • Collections & Iterators: Efficient data manipulation.
  • Abstraction without cost: Generics, Traits, and Trait Objects.
  • Smart Pointers: `Box`, `Rc`, and `Arc`.
  • Bare-Metal Programming: Understanding the `no_std` environment.
  • Hardware Abstraction Layers (HAL) and Peripheral Access Crates (PAC).
  • Interfacing with Hardware: GPIO, UART, SPI, I2C.
  • Concurrency in Embedded Systems: Interrupts and Safe Abstractions.
  • Real-Time Systems: Introduction to RTIC (Real-Time Interrupt-driven Concurrency).
  • Memory Management: Allocators and memory-constrained environments.
  • Writing and using device drivers in Rust.
  • Automotive Standards Overview: ISO 26262 (Functional Safety) & AUTOSAR.
  • In-Vehicle Networking: CAN Bus communication with Rust.
  • Project: Building a simple ECU (Electronic Control Unit) application.
  • Debugging and Testing Embedded Rust: Probes, GDB, and unit testing.
  • Performance Optimization for resource-constrained devices.
  • Capstone Project: A complete system (e.g., sensor data logger, dashboard controller).
  • Course Review and Q&A session.
  • Exploring the Rust Embedded Ecosystem.
  • Career pathways and further learning resources.
