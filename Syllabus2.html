Rust Programming - 20 Days Plan

Below is a detailed table of Rust programming topics and assignments for 20 days. Each day includes a topic, a detailed description, and an assignment to reinforce learning.

Day Topic Description Assignment
Day 1 Introduction to Rust
  • What is Rust? Overview of its features (memory safety, concurrency, performance).
  • Installing Rust using rustup.
  • Setting up the development environment (VS Code with Rust extensions).
  • Writing and running a simple "Hello, World!" program.
  • Install Rust and set up your development environment.
  • Write a "Hello, World!" program and run it using cargo run.
Day 2 Variables and Data Types
  • Understanding variables, mutability, and shadowing.
  • Primitive data types: integers, floats, booleans, characters.
  • Compound data types: tuples and arrays.
  • Type annotations and type inference.
  • Write a program that declares and prints variables of different data types.
  • Create a tuple and an array, and access their elements.
Day 3 Control Flow
  • Conditional statements: if, else if, else.
  • Looping constructs: loop, while, for.
  • Pattern matching with match expressions.
  • Write a program that uses if-else to determine if a number is even or odd.
  • Use a for loop to print numbers from 1 to 10.
  • Create a match expression to handle different cases for a variable.
Day 4 Functions
  • Defining and calling functions.
  • Function parameters and return values.
  • Understanding scope and lifetime.
  • Closures and their differences from functions.
  • Write a function that takes two numbers as input and returns their sum.
  • Create a closure that captures a variable from its environment.
Day 5 Ownership and Borrowing
  • Understanding Rust's ownership model.
  • Rules of ownership: move, copy, and borrow.
  • References and borrowing: mutable and immutable references.
  • Lifetimes and their role in memory safety.
  • Write a program that demonstrates ownership transfer.
  • Create a function that borrows a variable and modifies it using a mutable reference.
Day 6 Structs and Enums
  • Defining and using structs to group related data.
  • Tuple structs and unit structs.
  • Defining and using enums with variants.
  • Pattern matching with enums.
  • Create a struct to represent a rectangle and calculate its area.
  • Define an enum to represent different types of shapes and use pattern matching to handle them.
Day 7 Error Handling
  • Understanding the Result and Option types.
  • Handling errors using match and unwrap.
  • Propagating errors with the ? operator.
  • Creating custom error types.
  • Write a function that returns a Result type and handle the error using match.
  • Create a custom error type and use it in a function.
Day 8 Collections
  • Vectors: dynamic arrays in Rust.
  • Strings: String and &str.
  • Hash maps: storing key-value pairs.
  • Iterating over collections.
  • Create a vector of integers and perform operations like push, pop, and iteration.
  • Write a program that uses a hash map to count the frequency of words in a string.
Day 9 Modules and Packages
  • Organizing code using modules.
  • Creating and using crates.
  • Understanding the module hierarchy.
  • Using external crates from crates.io.
  • Create a multi-module project and use items from different modules.
  • Add an external crate to your project and use its functionality.
Day 10 Concurrency
  • Understanding threads and concurrency in Rust.
  • Using the std::thread module.
  • Message passing with channels.
  • Shared-state concurrency with Mutex and Arc.
  • Write a program that spawns multiple threads and prints messages from each thread.
  • Create a program that uses channels to communicate between threads.
Day 11 Traits and Generics
  • Defining and implementing traits.
  • Using traits for polymorphism.
  • Generics: writing reusable code.
  • Trait bounds and where clauses.
  • Define a trait and implement it for multiple types.
  • Write a generic function that works with any type implementing a specific trait.
Day 12 Advanced Ownership
  • Understanding smart pointers: Box, Rc, and Arc.
  • Interior mutability with RefCell.
  • Working with lifetimes in complex scenarios.
  • Write a program that uses Box to store data on the heap.
  • Create a program that demonstrates shared ownership using Rc.
Day 13 File I/O
  • Reading and writing files in Rust.
  • Working with directories and paths.
  • Error handling in file operations.
  • Write a program that reads a file and prints its contents.
  • Create a program that writes data to a new file.
Day 14 Testing
  • Writing unit tests and integration tests.
  • Using the #[test] attribute.
  • Test-driven development (TDD) in Rust.
  • Write unit tests for a function that calculates the factorial of a number.
  • Create an integration test for a multi-module project.
Day 15 Macros
  • Understanding declarative and procedural macros.
  • Writing custom macros using macro_rules!.
  • Using macros for code generation.
  • Write a macro that generates a function to print a message.
  • Create a macro that simplifies repetitive code patterns.
Day 16 Async Programming
  • Understanding asynchronous programming in Rust.
  • Using async and await.
  • Working with the tokio runtime.
  • Write an asynchronous function that fetches data from a URL.
  • Create a program that runs multiple async tasks concurrently.
Day 17 Unsafe Rust
  • Understanding unsafe code and its use cases.
  • Dereferencing raw pointers.
  • Calling unsafe functions and implementing unsafe traits.
  • Write a program that uses raw pointers to manipulate memory.
  • Create an unsafe function and call it from safe code.
Day 18 FFI (Foreign Function Interface)
  • Calling C functions from Rust.
  • Exposing Rust functions to C.
  • Using the libc crate.
  • Write a Rust program that calls a C function.
  • Create a Rust library that can be used from C.
Day 19 Web Development with Rust
  • Introduction to web frameworks like Rocket and Actix.
  • Building a simple web server.
  • Handling HTTP requests and responses.
  • Create a simple web server using Rocket or Actix.
  • Write a route that returns a JSON response.
Day 20 Building a Project
  • Planning and structuring a Rust project.
  • Using external crates and dependencies.
  • Building and publishing a crate to crates.io.
  • Build a small project (e.g., a CLI tool or a library) using everything you've learned.
  • Publish your project to crates.io (optional).
